1. Introduction

At PassMind Chrome Extension ("Extension", "We", "Us"), we prioritize user privacy. This Privacy Policy describes how PassMind Chrome Extension collects, uses, and protects data.

By using this extension, you agree to the practices described in this Privacy Policy.

2. Data Collection and Processing

2.1 Collected Data

PassMind Chrome Extension does not collect, store, or process any personal data.

The extension does NOT collect or process:

  • Passwords or credentials
  • Personal identification information (PII)
  • Browsing history
  • Location data
  • Device information

2.2 Data Transmission

The extension:

  • Does not send data to any server
  • Does not make API calls
  • Does not share data with third-party services
  • Does not use any data storage mechanisms (localStorage, sessionStorage, chrome.storage, etc.)
  • Does not collect telemetry or analytics data

3. How It Works

The extension works entirely locally in the user's browser:

3.1 Technical Operation

  • Only runs in the active browser tab
  • Only detects password input fields (<input type="password">)
  • Shows a button in the corner of the screen when password fields are detected
  • Opens a drawer panel to show/hide all password fields at once
  • When the button is clicked, only changes the input's type attribute
  • All operations are performed locally in the browser
  • No data is transmitted over the network

3.2 Permissions

The extension uses only the following minimal permissions:

  • activeTab: To work only in the active tab
  • storage: To store user settings (optional, based on user preference)

4. Tracking and Analytics

PassMind Chrome Extension:

  • Does not use tracking cookies
  • Does not use third-party analytics tools (Google Analytics, etc.)
  • Does not track user behavior
  • Does not collect data for advertising or marketing

5. External Links and Third-Party Services

The extension:

  • Does not connect to external resources
  • Does not load external sources (CDN, API, etc.)
  • Can work completely offline

6. Data Security

Since the extension collects no data, there is no risk of data breach. All operations occur locally on the user's device and no data is transmitted over the network.

7. Children's Privacy

PassMind Chrome Extension does not knowingly collect data from children. The extension is available for general use and no additional permission is required for users under 13, as no data is collected.

8. Privacy Policy Changes

We may update this Privacy Policy from time to time. For significant changes, we will publish the effective date of the new version on this page. We recommend checking this page regularly.
